## Overview
This repository contains the full Python pipeline for analyzing vibration-related telemetry using the AI4I Predictive Maintenance Dataset.  

## Structure
- `main.py`: Core pipeline (ingestion, cleaning, analysis, visualization).
- `requirements.txt`: Required libraries.
- `data/`: Original and cleaned datasets.
- `outputs/`: Static plots and animations.
- `README.md`: Documentation.

## Run Instructions
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Place the Kaggle dataset in `data/dataset_original.csv`.
4. Run: `python main.py`
5. Outputs will be saved in the `outputs/` folder.
